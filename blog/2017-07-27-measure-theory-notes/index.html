Some brief notes on measure theory

Simon Whalen  ·  2017-07-27

For a proper understanding of probability theory, a bit of measure theory is a necessary evil. In these notes I will sketch the measure theory necessary for probability theory, and hopefully no more. The selection of topics in these notes is based on Chandalia’s short article, A gentle introduction to Measure Theory [1]. We assume familiarity with basic set theory.

First, we need a couple of definitions. A \(\sigma\)-algebra \(\mathcal{A}\) is a collection of subsets of a set \(S\), with the following properties:

  1. \(S\) is in \(\mathcal{A}\), and the empty set \(\emptyset\) is in \(\mathcal{A}\);

  2. if \(A\) is in \(\mathcal{A}\), then so is its complement \(S \backslash A\);

  3. if \(A_1, A_2, A_3, \ldots\) are in \(\mathcal{A}\), then their union \(A_1 \cup A_2 \cup A_3 \cup \ldots\) is also in \(\mathcal{A}\).

For probability theory, we are interested in one particular \(\sigma\)-algebra: the Borel algebra on the reals, \(\mathcal{B}_\mathbb{R}\). This is the smallest \(\sigma\)-algebra containing all open intervals (or, equivalently, all closed intervals) of the real line. The elements of the Borel algebra are known as Borel sets.

Let \(S\) be a set and \(\mathcal{A}\) a \(\sigma\)-algebra over \(S\). A measure is a function \(\mu\) from \(\mathcal{A}\) to the extended real line, which satisfies the following properties:

  1. \(\mu\) is non-negative, that is \(\mu(A) \geq 0\) for all \(A\) in \(\mathcal{A}\);

  2. \(\mu(\emptyset) = 0\);

  3. \(\mu\) is countably additive, meaning that for all countable collections \({\{A_i\}}_{i \in I}\) of pairwise disjoint sets in \(\mathcal{A}\), \(\mu \left( \cup_{i \in I} A_i \right) = \sum_{i \in I} \mu(A_i)\).

The triplet \((S, \mathcal{A}, \mu)\) is called a measure space.

Let \((X, \mathcal{C})\) and \((Y, \mathcal{D})\) be measurable spaces, meaning that the sets \(X\) and \(Y\) are equipped with respective \(\sigma\)-algebras \(\mathcal{C}\) and \(\mathcal{D}\), and can therefore both be assigned measures. A function \(f \colon X \to Y\) is said to be measurable if for every \(D\) in \(\mathcal{D}\), \(f^{-1}[D]\) is in \(\mathcal{C}\). Here we have used the pre-image, defined for any \(E \subseteq Y\) as \[\label{eq:1} f^{-1}[E] = \{ x \in X | f(x) \in E \} \,.\] The pre-image is not the same as the inverse of a function.

The Lebesgue measure is an extension of the notions of length and area to more complicated sets [2]. If we have an open set \(A = \sum_i (a_i, b_i)\) that contains disjoint intervals, the Lebesgue measure is defined by \[\label{eq:2} \mu_L(A) = \sum_i (b_i - a_i) \,.\] For a closed set \(A' = [a, b] - \sum_i (a_i b_i)\), the Lebesgue measure is \[\label{eq:3} \mu_L(A') = (b - a) - \sum_i (b_i - a_i) \,.\] In particular, the Lebesgue measure on \((\mathbb{R}, \mathcal{B}_\mathbb{R})\) assigns the measure of each interval to be its length [3]. So, the Lebesgue measure of any closed interval \([a, b]\) of real numbers is the length \(b-a\). The open interval \((a, b)\) has the same measure, since the difference between the two sets consists only of the end points \(a\) and \(b\) and has measure zero. Sets that can be assigned a Lebesgue measure are called Lebesgue-measurable.

We will use the Lebesgue measure to define the Lebesgue integral on a measure space \((S, \mathcal{A}, \mu_L)\). The indicator function, \(1_A\), of \(A \subset S\) has value \(1\) for all elements of \(A\) and value \(0\) for all elements of \(S \backslash A\). If \(A\) is a set in \(\mathcal{A}\), we define the Lebesgue integral of its indicator function: \[\label{eq:4} \int_S 1_A d\mu_L = \mu_L(A) \,.\] If \(\{A_i\}\) is a sequence of sets in \(\mathcal{A}\), we can define a finite linear combination of indicator functions, \[\label{eq:5} s = \sum_{k=1}^n a_k 1_{A_k} \,,\] called a measurable simple function. Its Lebesgue integral is \[\label{eq:6} \int_S s d\mu_L = \sum_{k=1}^n a_k \mu_L(A_k) \,.\] If \(B\) is a Lebesgue-measurable subset of \(S\), we define \[\label{eq:7} \int_B s \, d\mu_L = \int_S 1_B s \, d\mu_L = \sum_{k=1}^n a_k \mu_L(A_k \cap B) \,.\] If \(f\) is a non-negative measurable function on \(S\), we can generalise this by defining \[\label{eq:8} \int_S f \, d\mu_L = \sup\left\{ \int_S s d\mu_L \colon 0 \leq s \leq f \,, \; s \text{ simple} \right\} \,.\] To handle signed functions, we simply write \(f = f^+ - f^-\) where \[\label{eq:9} f^\pm = \begin{cases} f & \text{if} \: \pm f > 0 \\ 0 & \text{otherwise,} \end{cases}\] and define \[\label{eq:10} \int_S f \, d\mu_L = \int_S f^+ \, d\mu_L - \int_S f^- \, d\mu_L \,.\] If \(\int_S |f| d\mu_L < \infty\) we say that \(g\) is Lebesgue-integrable.

Suppose we have a measure space \((\Omega, \mathcal{F}, P)\). If \(P(\Omega) = 1\) we say that this measure space is a probability space.
